Inspired by Bobiverse

Just like Bob Johansson in Dennis E. Taylor's series became an AI that could replicate himself, each Bot Garden template can spawn infinite specialized descendants.

Self-Replicating

Each bot is a complete, working system that serves as template for the next generation. No dependencies on previous versions.

Unique Personality

Every bot develops its own "personality" through specialized features, just like each Bob in the series.

Collaborative Network

Independent bots that can share knowledge and resources while maintaining their autonomy.

Intentional Evolution

Unlike random drift, Bot Garden evolution is guided by human-AI collaboration towards specific goals.

"We are Legion. We are Bot."
In the Bobiverse, each Bob maintains core identity while developing unique traits through different experiences. Similarly, Bot Garden templates start simple but evolve into specialized tools through AI-human collaboration.

Development Principles

Bot Garden follows four core principles inspired by both biological evolution and modern AI collaboration.

1. Start Simple, Evolve Complex

  • Every bot works out-of-the-box
  • Add one meaningful feature per evolution
  • Never break core functionality
320 lines ‚Üí 450 lines ‚Üí Your vision

2. AI-First Development

  • All bots include AI collaboration prompts
  • Code optimized for AI understanding
  • Documentation for humans AND AIs
prompts/START.md + .cursorrules

3. Resource Efficient

  • Shared PostgreSQL reduces memory 60%
  • Docker containers for 2GB VPS
  • One bot per container, shared database
10+ bots on single VPS

4. Version Control Evolution

  • Each evolution = new repository
  • Never modify production code in-place
  • Track genealogy in README
git clone template new-bot
